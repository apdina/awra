# Avatar Images

This directory contains avatar images for the application.

## Structure

- `basic/` - 9 basic avatars (3 genders × 3 age groups)
  - young_mas.png
  - young_fem.png
  - young_between.png
  - middle_mas.png
  - middle_fem.png
  - middle_between.png
  - old_mas.png
  - old_fem.png
  - old_between.png

- `special/` - 10 special avatars
  - sp1.png through sp10.png

## Image Requirements

- Format: PNG with transparency
- Size: 256x256 pixels (or larger, will be scaled)
- File naming: Must match the avatar names in `convex/avatar.ts`
